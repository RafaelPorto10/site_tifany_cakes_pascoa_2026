# 🍰 Cardápio Digital

## 📌 Sobre o projeto
Este projeto foi desenvolvido para uma loja de doces, com o objetivo de apresentar os produtos de forma organizada e visualmente atrativa para os clientes.

## 🚀 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 🎯 Objetivo
Criar um cardápio digital simples, funcional e fácil de navegar.
