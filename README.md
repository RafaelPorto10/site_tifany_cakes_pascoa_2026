# 🍰 Tiffany Cakes - Cardápio Digital Páscoa 2026

## 📌 Sobre o projeto
Este projeto foi desenvolvido com o objetivo de simular um site de uma confeitaria, apresentando um cardápio digital temático de Páscoa.

A aplicação permite visualizar produtos, categorias e informações de forma organizada e visualmente atrativa.

## 🚀 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 🎯 Funcionalidades
- Exibição de produtos
- Layout responsivo
- Interface voltada para experiência do usuário
- Organização por categorias
